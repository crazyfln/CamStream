# CamStream
Stream your webcams over http using mjpeg format

###### [*] Basic authentication [admim,admin]
###### [*] Face detection
###### [*] Multiple webcams
###### [*] Image compression (quality, crop, grayscale)
###### [*] example url: http://localhost:8080/?cam=0&q=50&gray=true&crop=2
